## Installation

```bash
$ npm install
```

## Running the app

```bash
# 로컬 환경에서 구동시
$ npm run dev

# 운영환경에서 구동시 빌드 후 구동
$ npm run build
$ npm run prod

# PM2로 구동시
$ npm run build
$ pm2 start dist/main.js --name be-memberportal
```



## 환경파일
